# DeepHomieJS
A discord bot meant for my friend group that streams music from Youtube
He also does dice rolls, makes choices, and tells a lot of inside jokes.

To use, create a config.json file with 
```
{
  "token": <Token>,
  "clientId": <Client id>,
  "apiKey": <Youtube Api key>
 }
```
